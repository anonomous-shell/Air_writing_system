# Air Writing System ✍️

A real-time Air Writing Recognition System built using **Python**, **OpenCV**, and **MediaPipe**. This project tracks hand and finger movements through a webcam and allows users to write in the air without touching any physical surface.

## 🚀 Features

* Real-time hand tracking using MediaPipe
* Air writing with finger gestures
* Smooth drawing and trajectory tracking
* Computer vision-based interaction
* Lightweight and easy to run

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/anonomous-shell/Air_writing_system.git
cd Air_writing_system
```

### 2. Create a virtual environment (optional)

```bash
python -m venv venv
```

Activate the virtual environment:

**Windows**

```bash
venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the project

```bash
python main.py
```
